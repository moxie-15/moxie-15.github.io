Overview

This is my first complete desktop application built with Flutter and Firebase. The CBT app includes:

  • Start Screen
  • Login / Sign-in (Firebase Auth + REST API)
  • Sign-up / Registration Screen
  • Student Dashboard
  • Teacher Dashboard
  • Admin Dashboard
  • Quiz List / Selection Page
  • Quiz Page (taking quizzes)
  • Quiz Results / Review Page
  • Submission Confirmation Page
  • Logout Page
